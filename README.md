# 🐕 Simple Robot Dog Mechanical Design

## Overview
This project presents the initial mechanical design of a simple quadruped (robot dog) for the Smart Methods assignment.

## Project Objectives
- Design the robot body (chassis)
- Design the legs
- Determine the Degrees of Freedom (DOF)
- Select suitable servo motors
- Perform an initial torque calculation
- Analyze stability and center of gravity
- Propose a walking gait
- Discuss expected mechanical challenges
